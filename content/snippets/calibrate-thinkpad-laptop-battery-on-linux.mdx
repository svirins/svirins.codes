#### Install `tlp`

```shell
sudo add-apt-repository ppa:linrunner/tlp
sudo apt update
sudo apt install tlp tlp-rdw --no-install-recommends
sudo apt install acpi-call-dkms
sudo tlp-stat
sudo apt install powertop
sudo powertop -c
```
